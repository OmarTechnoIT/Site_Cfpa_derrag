# Centre de Formation et d’Apprentissage

Site web professionnel dédié à la présentation des formations, services pédagogiques et activités du centre de formation.

## Description

Cette plateforme permet de :

- Présenter les formations disponibles
- Faciliter l’inscription des étudiants
- Publier des actualités et annonces
- Présenter le centre et les formateurs

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## Fonctionnalités

- Interface moderne et responsive
- Gestion des utilisateurs
- Formulaire d’inscription
- Gestion des formations
- Galerie d’images
